# py学习的一些小的demo
跟着[b站视频](https://www.bilibili.com/video/BV1Yh411o7Sz)做的

爬虫简单过程:
 1. 找到指定url
 2. 发起请求
 3. 获取响应数据
 4. 持久化存储

数据解析:

1.  聚焦爬虫:爬取页面中指定的页面内容.
    1.  指定url
    2.  发起请求
    3.  获取相应数据
    4.  数据解析
    5.  持久化存储  

2.  数据解析分类
    1.  正则
    2.  bs4
    3.  Xpath



数据解析原理概述:

1.  解析的局部文本内容都会在标签之间或标签对应的属性进行存储
2.  对指定标签的定位
3.  标签或者标签对应的属性中的存储的数据值进行解析
